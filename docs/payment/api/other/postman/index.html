Thiết Lập Postman

Postman Collection giúp bạn nhanh chóng hiểu và test được quy trình thanh toán. Đây là công cụ hỗ trợ đắc lực khi tích hợp, khiến cho các API sử dụng cho tích hợp trở nên dễ hiểu hơn.

Sử dụng Postman Collection​

Tải xuống và thiết lập Postman tại đây.
Nhập postman collection sẵn có cho những luồng khác nhau.
Lấy Partner Code, Access Key, và Secret Key của bạn tại website MoMo cho Doanh nghiệp
Cập nhật các giá trị này theo những biến trong môi trường bạn đang thực hiện (Testing/Production):
• paygate : https://test-payment.momo.vn cho môi trường Testing và https://payment.momo.vn/ cho môi trường Production
• partnerCode : Partner Code của bạn ở bước 3
• accessKey : Access Key của bạn ở bước 3
• secretKey : Secret Key của bạn ở bước 3
