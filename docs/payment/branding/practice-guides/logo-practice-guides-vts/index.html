Logo Ví Trả Sau

Introduction​

In addition to the requirement for using the MoMo logo, if your payment options include Ví Trả Sau on MoMo, please add the Ví Trả Sau logo accordingly.

For One-time Payments​

For those who wish to demonstrate the Ví Trả Sau option on MoMo (payment page, payment settings, etc.), you need to:

  • Always display the official MoMo logo. This is a mandatory element of our brand identity and must be shown whenever Ví Trả Sau is presented.
  • If Ví Trả Sau is available, add the logo next to or under the MoMo logo. Do not replace or omit the MoMo logo under any circumstance.
  • Show brand name of this payment mode. To make it clearer, a description attached is encouraged.

Our brand is properly written as Ví Trả Sau trên MoMo. Using the right brand name in all interfaces and documents, contracts,... is a way to show your respect for us, just as we do for our partners.

1. In a given context: “Select a payment method” is available

Horizontal layoutVertical layout

Once your checkout page clearly shows "Select the payment method":

  • Only need to use Ví Trả Sau trên MoMo to illustrate the back of logo.
  • In case there is space for a description near the Ví Trả Sau logo, please use the phrase Không lãi suất đến 45 ngày, mở 3 phút (in Vietnamese) or Up to 45 days interest-free, 3-minute signup (in English).
  • In case of online payment using ATM card (Napas) powered by MoMo, please use the phrase Thẻ ATM Nội Địa (in Vietnamese) or Domestic ATM Card (in English).
  • In case of online payment using VISA/Master/JCB card (Cybersouce) powered by MoMo, please use the phrase Thẻ VISA/Master/JCB (in Vietnamese) or VISA/Master/JCB Card (in English).

2. Without a given context: “Select a payment method” is not available

Horizontal layoutVertical layout

When the phrase Select a payment method is not available, for each option, please specify the type of "Pay with...".
For example, use the phrase Thanh toán với Ví Trả Sau trên MoMo (in Vietnamese) or Pay with Ví Trả Sau on MoMo (in English).

In case there is space for a description near the Ví Trả Sau logo, please use the phrase Không lãi suất đến 45 ngày, mở 3 phút (in Vietnamese) or Up to 45 days interest-free, 3-minute signup (in English).

3. Other acceptable practice

Rectangle iconLogo on the back of the word

In case there is space for a description near the Ví Trả Sau logo, please use the phrase Không lãi suất đến 45 ngày, mở 3 phút (in Vietnamese) or Up to 45 days interest-free, 3-minute signup (in English).

For Tokenization Payments​

If the payment models integrated are tokenization, you can follow the above guides, or use below best practices to inform users better:

1. Before binding

Regular use
With monotone logo, the “not yet set-up” payment method is displayed better

In case there is space for a description near the Ví Trả Sau logo, please use the phrase Không lãi suất đến 45 ngày, mở 3 phút (in Vietnamese) or Up to 45 days interest-free, 3-minute signup (in English).

2. After successfully binding

Brand name should be showed with 4-digit phone number

In case there is space for a description near the Ví Trả Sau logo, please use the phrase Không lãi suất đến 45 ngày, mở 3 phút (in Vietnamese) or Up to 45 days interest-free, 3-minute signup (in English).
