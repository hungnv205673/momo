Logo MoMo

Introduction​

In addition to the requirement for using the logo MoMo, please refer to the instructions below in order to best demonstrate the payment with MoMo on your website/app.

For One-time Payments​

For whom desire to demonstrate the payment method with MoMo (payment page, payment settings page,...), you need to:

  • Separate payment methods powered by MoMo from others (you will want to define the correct type of payment method the customer is looking for to make an appropriate order).
  • Use logo provided by MoMo under any circumstances. This is the primary element in our brand identity.
  • Show brand name of this payment mode. To make it clearer, a description or a badge attached is encouraged.

Our brand is properly written as MoMo, with two capitals "M" and two lowercases "o". Any other written forms such as MOMO, momo, Momo,... are not acceptable. Using the right brand name in all interfaces and documents, contracts,... is a way to show your respect for us, just as we do for our partners.

1. In a given context: “Select a payment method” is available

Horizontal layoutVertical layout

Once your checkout page clearly shows "Select the payment method":

  • Only need to use MoMo to illustrate the back of MoMo's logo.
  • In case there is space for a description near the MoMo logo, please use the phrase Linh hoạt Trả ngay hoặc Trả sau (in Vietnamese) or Flexible: Pay now or Pay later (in English).
  • In case of online payment using ATM card (Napas) powered by MoMo, please use the phrase Thẻ ATM Nội Địa (in Vietnamese) or Domestic ATM Card (in English).
  • In case of online payment using VISA/Master/JCB card (Cybersouce) powered by MoMo, please use the phrase Thẻ VISA/Master/JCB (in Vietnamese) or VISA/Master/JCB Card (in English).

2. Without a given context: “Select a payment method” is not available

Horizontal layoutVertical layout

When the phrase Select a payment method is not available, for each option, please specify the type of "Pay with...".
For example, use the phrase Thanh toán bằng MoMo (in Vietnamese) or Pay with MoMo (in English).

In case there is space for a description near the MoMo logo, please use the phrase Linh hoạt Trả ngay hoặc Trả sau (in Vietnamese) or Flexible: Pay now or Pay later (in English).

3. Other acceptable practice

Rectangle iconLogo on the back of the word

In case there is space for a description near the MoMo logo, please use the phrase Linh hoạt Trả ngay hoặc Trả sau (in Vietnamese) or Flexible: Pay now or Pay later (in English).

For Tokenization Payments​

If the payment models integrated are tokenization, you can follow the above guides, or use below best practices to inform users better:

1. Before binding

Regular use
With monotone logo, the “not yet set-up” payment method is displayed better

In case there is space for a description near the MoMo logo, please use the phrase Linh hoạt Trả ngay hoặc Trả sau (in Vietnamese) or Flexible: Pay now or Pay later (in English).

2. After successfully binding

Brand name should be showed with 4-digit phone number

In case there is space for a description near the MoMo logo, please use the phrase Linh hoạt Trả ngay hoặc Trả sau (in Vietnamese) or Flexible: Pay now or Pay later (in English).
